## üöÄ Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hetari/shadcn-docs-template.git
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

### Development

Start the development server:

```bash
bun dev
```

The site will be available at `http://localhost:3000`.
